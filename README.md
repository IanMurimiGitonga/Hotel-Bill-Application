# Hotel Bill Apllication
This project is an application is for a restaurant or a cafe.<br/>
It takes in customer orders and provide receipts to the customers.

## Features
- Menu: Displays the available delicacies.<br/>
- Tip-Section: Allows the customer to decide how much tip to give.<br/>
- Receipt: Shows the order made,time stamp and amounts.<br/>

## Technologies Used
- Python
